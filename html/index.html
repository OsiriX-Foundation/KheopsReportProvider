Kheops Report Provider

How create a report provider on Kheops

  1. Create an album or go on your favorite album
  2. Go to Setting
  3. Go to Report providers
  4. Click on New report provider
  5. Choose a name
  6. Add this configuration url: http://129.194.108.70/.well-known/kheops-report-configuration

If you follow this step, you can now use a report provider for a study in your album !

On the menu Report providers in the setting of your album, you can always edit or check the state of the report provider.

How use a report provider on Kheops

  1. Choose a study
  2. Click on
  3. And choose your report provider
